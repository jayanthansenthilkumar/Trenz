Web Weave Challenge 2025

April 30, 2025
09:00 AM - 4:00 PM
Individual/Team (max 2 members)

Rules & Guidelines

  • Event Name: Web-Designing
  • Overview: Unleash your creativity and technical prowess in this web design competition where innovation meets functionality. Teams will design a complete website based on a theme of their choice, within the allotted time.
  • Team Format: 4 members per team
  • Eligibility: Open to all departments
  • Duration: 4 hours
  • Theme: Choose your own theme
    (e.g., Portfolio, E-commerce, Educational, Non-profit, etc.)
  • Output Required: A complete static or dynamic website based on the selected theme
  • Tools Allowed:
    Visual Studio Code Sublime Text Figma GitHub
  • AI Assistance: AI-based tools (like ChatGPT, GitHub Copilot) are allowed for ideation and code suggestions
  • Judging Criteria:
    • Creativity and Originality
    • Responsiveness and Functionality
    • Code Quality and Structure
    • Aesthetic Design and User Experience
