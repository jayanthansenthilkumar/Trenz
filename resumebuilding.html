Built a Resume - Professional CV Competition

April 30, 2025
09:00 AM - 04:00 PM
Individual Participation

Rules & Guidelines

  • Name of the Event: Build a Resume
  • Event Overview: Build-A-Resume is a resume creation competition where participants will craft a professional resume on the spot using online tools. The competition is designed to test students' ability to present their skills and experiences effectively and professionally.
  • Participation Format:
    • Individual participation only
    • Resumes must be created during the event time using online or offline tools
  • Event Structure:
    • Participants will be given a duration of 1.5 to 2 hours to create their resume from scratch
    • Resume can be created using online Platforms
    • At the end of the session, participants must export their resume in PDF format and submit it to the organizers
  • Clarification on Tools and Rules:
    • Participants must be physically present at the event venue
    • Resume must be created during the event duration only
    • Pre-prepared or printed resumes are not allowed
    • Copying from sample resumes or using AI-generated content is prohibited
    • The resume must be created during the event timeframe
    • The resume should not exceed two page (for students/freshers) unless otherwise specified
    • File name format: FullName_Resume.pdf can be send to the email Id which will be shared in the venue
